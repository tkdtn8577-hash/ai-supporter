# 인스타그램 인플루언서 분석 도구 — API 선택 리포트

**작성일:** 2026-07-05  
**작성 목적:** Facebook 개발자 계정 없이 인플루언서 분석 도구를 운영하기 위한 API 선택 근거 문서

---

## 1. 배경 — 왜 Facebook API를 쓸 수 없는가

초기 구현은 Meta의 공식 **Instagram Graph API (Business Discovery)** 를 사용하도록 설계되었습니다.  
이 API를 사용하려면 반드시 **Facebook 개발자 계정 등록**이 필요하지만, 수 주째 다음 오류로 등록이 막혀 있습니다.

> *"현재 이 변경 사항을 적용할 수 없습니다. 평소에 사용하지 않는 기기를 사용 중이신 것으로 확인되어 계정 안전을 위한 조치를 적용해야 합니다."*

Facebook 측의 계정 보안 정책으로 인한 제한이며, 사용자가 직접 해제할 수 없는 상황입니다.  
따라서 **Facebook 계정 없이 동일한 기능을 제공하는 서드파티 API** 로 전환합니다.

## 3. 최종 선택: HikerAPI

### 선택 근거

**① 무료 한도 2배**  
HikerAPI는 가입 즉시 100회 무료 요청을 제공합니다 (SociaVault는 50회).

**② 이후 단가 8배 저렴**  
무료 한도 소진 후 추가 사용 시 HikerAPI는 $0.0006/회인 반면, SociaVault는 $0.005/회입니다.  
1,000회 사용 기준: HikerAPI $0.60 vs SociaVault $5.00

**③ 문서화 우수**  
Swagger/ReDoc 기반 완전한 API 문서 제공으로 유지보수가 쉽습니다.

**④ 필요 데이터 전부 제공**  
팔로워 수, 바이오, 게시물 수, 최근 게시물의 좋아요·댓글 수, 프로필 사진 URL — 분석에 필요한 모든 필드를 반환합니다.

## 4. 무료 한도 계산

| 작업 | API 호출 수 |
|------|------------|
| 인플루언서 1명 프로필 조회 | 1회 |
| 인플루언서 1명 게시물 조회 | 1회 |
| **10명 전체 분석 1세션** | **20회** |

- 무료 100회 ÷ 20회/세션 = **최소 5회 전체 분석 가능**
- 월 2~3회 분석 기준: 무료 한도 내에서 약 1~2개월 사용 가능
- 이후 충전: $5 충전 시 약 8,300회 추가 요청 → 실질적으로 무제한에 가까움

## 5. 사용 방법 (초기 1회 설정)

### Step 1 — HikerAPI 가입

1. [https://hikerapi.com](https://hikerapi.com) 접속
2. **Sign Up** 클릭 → 이메일·비밀번호 입력 (신용카드 불필요)
3. 이메일 인증 완료

### Step 2 — API 키 발급

1. 로그인 후 대시보드에서 **API Key** 복사

### Step 3 — 환경 변수 등록

**로컬 개발 (`folio/.env.local`):**
```
HIKERAPI_KEY=발급받은_API_키
```

**Vercel 프로덕션:**
1. Vercel 대시보드 → 프로젝트 선택 → Settings → Environment Variables
2. `HIKERAPI_KEY` 추가 (값: 발급받은 API 키)
3. **Redeploy** 실행

### Step 4 — 분석 시작

`/admin/instagram` 접속 → 카테고리 + 계정명 입력 → **분석 시작** 클릭

## 6. 기술 구현 방식

API 키 보안을 위해 HikerAPI 호출은 **서버 사이드(Next.js API Route)**에서만 실행됩니다.

```
브라우저 → GET /api/instagram/influencer?username=xxx
              ↓ (서버)
         HikerAPI 프로필 조회 (HIKERAPI_KEY 사용)
              ↓
         HikerAPI 게시물 조회
              ↓
         정제된 데이터 반환
브라우저 ← { followers, media, biography, ... }
              ↓
         채점 (참여율·카테고리·활동성·팔로워 점수)
              ↓
         결과 표시
```

채점 로직은 기존 Formula를 그대로 유지합니다:

```
종합 점수 = 참여율(40%) + 카테고리 적합도(30%) + 활동성(15%) + 팔로워 적정성(15%)
```

## 7. 한계 및 주의사항

- **비공식 API**: HikerAPI는 Meta의 공식 파트너가 아닙니다. Instagram의 내부 구조 변경 시 일시적 오류가 발생할 수 있습니다 (통상 24~48시간 내 수정).
- **비공개 계정 분석 불가**: Instagram 비공개 계정은 어떤 API로도 조회할 수 없습니다.
- **요청 속도 제한**: 계정당 요청 간 600ms 지연을 적용하여 차단 방지.
- **데이터 정확도**: 좋아요 수 숨김 기능을 사용하는 계정은 `like_count = 0`으로 반환될 수 있습니다.
